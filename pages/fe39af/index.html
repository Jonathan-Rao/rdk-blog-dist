          独立开发者公司资质与税务工作.

          # 什么情况下需要公司资质？

          • 接入支付宝/微信支付 （个体户也可以）
          • 微信公众号/小程序的各种高级功能
          • 接入一些著名的广告平台（优量汇、穿山甲）
          • 申请游戏版号
          • 注册商标
          • 享受云厂商的企业服务
          • 上架一些国内应用商店（例如华为现在只允许企业身份上架应用）
          • ....

          如果你认为未来，你需要开展上述业务中的部分，那么你需要公司资质

          # 有哪些身份可以选？

          1. 个体户
          2. 个人独资企业
          3. 普通责任有限公司

          他们之间的详细区别和优劣，建议查看https://zhuanlan.zhihu.com/p/65148790 (opens new window)

          # 建议选啥

          我来讲讲经过收集情报后，为什么我认为独立开发者适合个人独资企业：

          首先排除个体户，现在有很多地方个体户身份都不认了，比如广告平台，比如华为应用商店，而且也必须要记账，既然要麻烦，还不如一步到位。

          其次来对比个人独资和普通责任有限公司：

          • 企业身份

          • 目前没有收集到情报说，这两种在被认定为"企业"上有什么区别。

          • 个人独资的名字不能叫XX公司，只能是XX中心、XX工作室等等，听起来可能没那么有气势。

          • 收税

          • 个人独资没有企业所得税，直接走个人的经营所得税。

          • 普通责任有限公司的收入要收企业所得税，打给个人走分红还要再算所得税。

          • 避税方式

          • 个人独资可以申请核定征收，用来避税（后面会举例）。

          • 普通责任有限公司，通常需要把各种支出算到公司头上（比如你买个ipad），然后抵掉你的收入，来做到避税。

          • 打钱给自己

          • 个人独资可以公帐直接打给自己，没有障碍。

          • 普通责任有限公司，只能通过分红、发工资等手段发钱给自己，而每种也都需要再过个人所得税。

          如果你能确定你只有一个人，那么个人独资企业无疑是最佳选择。实在不行，你可以再开一家公司嘛。

          个人独资也不是没有弊端的，主要有

          • 无限责任，也就是你不能破产，但是如果你只用来收钱避税，装个企业身份，也就不用担心这个问题。
          • 只能你一个人。

          # 缴税案例分析

          假设你今年通过卖APP收入10万。

          （案例里公司没有算增值税、附加税等乱七八糟我看不懂的，我理解反正都要交，而且是小头）

          # 纯个人身份

          你要交税的额度是

          10万*40% - 7千 = 3.3万 （收入额x税率 - 速算扣除数）

          是不是很淦？辛辛苦苦大半年才赚了10万，连一个初级码农都不到，还要被国家扣掉3万3

          # 个人独资

          个人独资可以走核定征收，具体的概念可以自行百度。这里简单解释一下，就是不让你算应纳税额了，默认你的利润率为一个固定值，例如10%（其他行业定的貌似都是10%）。

          你今年收入10万，你就不用算成本了，默认你的利润就是1万，也就是是说你的应纳税额就是1万。

          然后按照因为个人独资不交企业所得税，所以你可以直接开始算个人经营所得税

          你的应纳税额经过核定征收，额度为 10万*10%=1万

          所以你要交的钱就是 1万*5% = 500块

          只要500块，小朋友你是否有很多问号？

          # 普通责任有限公司

          这个地方其实是不好算的，因为应纳税额为 收入 - 成本

          你可以通过做账，把买的东西全算到公司头上，来甚至把成本做到跟收入一样，你就可以0申报啦。

          不过比如你这年的收入10万，你要把成本也算到10万，也是挺难的，我们假设你成功地算成了9万成本。

          那么应纳税额是1万

          首先要算企业所得税，一般企业是要交25%的，但因为你这收入太少了还只算小微企业，所以只有10%。

          所以你的企业所得税是 1万 * 10% = 1千

          别急，你还要把钱打给自己，普通责任有限公司不能直接公账给私账打款，所以你要走给自己分红的形式，分红的税率是20%

          所以你还要扣掉 （1万 - 1千）* 20% = 1800 （(纳税额 - 企业所得) * 分红税率）

          所以最后，你要缴税的额度是 2800

          # 总结

          所以在你独立开发年收入10万的情况下，你

          • 个人身份纳税3万3
          • 个人独资身份纳税500
          • 普通责任有限公司身份纳税2800

          不用我多说，你也知道怎么选了吧。至于年收入更高的情况下，哪个更划算，自己算一算吧。
