git使用问题汇总

# 1.合并分支容易出现的问题？

如果已经git merge过了，仅仅只是本地git状态不一致，但是git status 依然提示有很多commit未提 交，那么可以使用一下命令解决：

git fetch --all
git pull
git reset --hard origin/master
