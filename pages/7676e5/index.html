# 个人开工作室注册什么类型的公司好？ (opens new window)

# 为什么要开一家公司？如何开一家公司？

如果你看过财富自由之路 (opens new window)就会明白从法律意义上来说，开公司用公司身份缴税是成本最低的，而且从看到的材料来说年收入 10 万的情况下，你

● 个人身份纳税 3 万 3

● 个人独资身份纳税 500

● 普通责任有限公司身份纳税 2800

目前来说优先推荐独资（最大的问题是只能自己一个人承担无限责任，好处就是目前各大平台基本都承认这个公司身份给的接口权限都不会少，不像个体工商户都开始做限制，不是很承认公司身份）

另外推荐使用自己老家住宅信息，后边核查以及报税什么的都方便，也能省注册地费用（不然租地址要付费的，使用老家住宅一般回老家两三天就办下来了，各地政府都在推行一天领证，自己跑一般花不了啥钱，3 个公章 200；办理对公账户可能需要公司招牌就是类似牌匾，几十块，然后选择当地本地商业 (opens new window)银行 (opens new window)基本都是免费，然后你就可以用自己的公司给自己交社保，如果你是自由职业 (opens new window)可以省代交社保的钱）

跑腿流程 2 天（1 天拿执照 +1 天拿对公账户）跑腿之前，先打电话询问了解清楚需要哪些证明材料，可以省很多时间，另外第一次办理财 (opens new window)税大概率也需要跑腿，后续一般不需要了（综合时间可能需要 1 到 2 个月；综合花费一般就是自己跑腿 + 公章还有银行开户的钱，一般不超过 300）。

# 关于财税和代理记账问题：

不如使用自己记账服务

拓展推荐》阿里云领优惠券 (opens new window)

不是很在意有没有人工客服可以购买〉阿里云智能财税（不超过 360 一年），个人就可以简单自己记账报税，熟练的话一年后自己直接就能操作不需要买代记账这种服务了。 (opens new window)

如果比较在意人工客服，可以考虑费用高一些的：自记账：不用找代理记账，零基础自己记账报税 (opens new window)

自记账专为小微企业、个体户设计（全国通用）不找代理记账零基础自己记账报税，优势就是因为用他们这个除了税务疏漏可以全额赔付，另外就是人工客服好说话，注册公司有疑问，税务疑问都能找到客服咨询解决，而且是多端 App。处理起来税务环境局限比较小，很方便。

无论是通过阿里云的智能财税又或者是通过自记账，都是能省下来买代理记账的费用，综合下来养一个公司名头基本一年不会超过 1000 块钱吧，一般来说除了所得税也没什么特别需要缴纳的税务，个人以为你学习能力强一些，跟着智能财税自己学会了，以后可能连这个也不用买了，我一个相亲认识的朋友现在就是自己报税交社保啥的不需要别人了，也不用买服务了。

# 如果你的创业项目包括了网站等需要用到服务器的地方：

# 服务器服务：

比较建议基础版本直接先买一年再续费 5 年（2000 左右），折合下来一年成本 300 多，能用 5/6 年，你的互联网服务会长久一些，不会说推广几年不用了给别人做了免费推广了，而且一个项目发展起来需要时间才能看到结果，不是人人都很简单就能直接爆发增长一夜暴富财务自由的。

腾讯云服务器域名：点击注册购买 (opens new window)（99 一年）

阿里云服务器域名：点击注册购买 (opens new window)（99 一年）

# 托管性质服务：

如果你只是搞个网站，没必要买服务器，虚拟主机就可以，wordpress 建站就很好，用过很多程序后的感觉就是这样。

WPBOX：超便宜实惠的 wordpress 云托管平台，快速开启个人博客企业官网或者跨境电商外贸独立站吧 (opens new window)

快鱼主机：更适合家跨境电商的独立站建站虚拟主机服务 (opens new window)

Buckydrop:无货源全自动化履约跨境电商平台，快速入门跨境电商业务（也可以一键建站） (opens new window)

花生小店：免费的国内商城 saas 服务 (opens new window)

# 解决支付问题：

基本就是注册后集成到服务里，自己银行卡提现就可以了，这些平台也就首个通道手续费，一般都能接受
